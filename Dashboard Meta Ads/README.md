# Dashboard Meta Ads

Protótipo local para monitoramento consolidado de contas de anúncio da Meta.

## Executar

```powershell
node preview-server.js
```

Acesse `http://127.0.0.1:8091`.

Os dados atuais são demonstrativos. A integração futura deverá substituir o array `accounts` em `app.js` por uma API interna, mantendo tokens da Meta somente no servidor.
